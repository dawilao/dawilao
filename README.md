## 📦 Projetos principais

### [AgendaObras](https://github.com/dawilao/AgendaObras)
Sistema web para rastreamento de obras e demandas de engenharia. Controla contratos, gera checklists automáticos com prazos e dependências, gerencia medições mensais e envia alertas por e-mail nos dias úteis. Construído com Python + NiceGUI + SQLite.

### [Gerador de Requisições](https://github.com/dawilao/GeradorRequisicoes)
Aplicação desktop para geração automatizada de requisições de pagamento, aquisição e e-mails para fornecedores. Exporta para Excel, tem login por usuário e é distribuído como `.exe` para uso corporativo sem necessidade de Python instalado. Construído com CustomTkinter + SQLite + OpenPyXL.

### [PDFMaster](https://github.com/dawilao/PDFMaster)
Aplicativo desktop para criação e manipulação de PDFs com interface gráfica.

### [OrcamAprovado](https://github.com/dawilao/OrcamAprovado)
Ferramenta para controle de orçamentos aprovados.
